Um criador independente criou um sistema para a **extracção de bitcoins com um Nintendo GameBoy**. 

Um sistema que requer um pouco de hardware, um pouco de tempo para criar o programa e, o mais importante, um pouco de tempo. 

Porque o GameBoy oferece uma quantidade ridícula de poder computacional que torna o sistema divertido, mas não de todo eficiente. Ver o resultado em vídeo.

Há uma forte escassez de componentes electrónicos. Afecta muitas famílias de produtos: consolas de videojogos, placas gráficas, smartphones, televisões, etc. Isto leva a uma deterioração da disponibilidade dos produtos, é claro. 

E o bloqueio do Canal de Suez irá, evidentemente, agravar a situação. **As novas placas gráficas de Nvidia e AMD são duplamente afectadas**. Por um lado, há a falta de stock. E há a loucura da mineração da moeda criptográfica na outra.

Este negócio está em plena expansão. O aumento do valor de uma moeda criptográfica torna a exploração mineira particularmente atractiva, apesar dos elevados custos energéticos. 

Existem formas alternativas de minerar as moedas criptográficas? Sim. São rentáveis, eficientes, viáveis? Nem todos eles. Como prova, queremos mostrar-vos esta experiência feita com um Game Boy original (sim, a **antiga consola portátil Nintendo lançada em 1989**). Pode encontrar o vídeo completo no final deste artigo.

## O RAPAZ DO JOGO PODE MINAR, MAS É MUITO, MUITO LENTO

Durante este vídeo, que dura um pouco menos de 10 minutos, descobre tudo o que precisa para **transformar um Game Boy numa fábrica de mineração de moedas criptográficas, aqui Bitcoins**. 

O director deste vídeo precisava de um Raspberry Pi Pico para fazer a ligação entre o Game Boy e o PC (que faz a ligação com os nós do Bitcoin). 

Teve também de criar um cabo específico para transferir dados do Game Boy para o PC, bem como um cartucho em branco para fazer piscar o programa de mineração (que na realidade é bastante simples).

Depois de todo o sistema estar configurado, o programador liga a consola e... inicia a extracção de Bitcoins. 

Claro que, considerando o tamanho dos dados, a velocidade do processador e a velocidade de transmissão da informação, o Game Boy não é tão rápido como uma GeForce ou uma Radeon para a mina. 

Muito, muito (muito?) mais lento. De acordo com os cálculos do promotor, seriam necessários vários milhões de milhões de anos para extrair... 1 Bitcoin (ou vários quadriliões americanos, que é o mesmo que o nosso bilião). 

Nessa altura, até mesmo o sol estará morto (a sua esperança de vida é de 5,5 mil milhões de anos).
